Simulating a zoo with lions and fish

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You are given a task to simulate a Zoo. Your job is to create an abstract class Animal that has name and age as properties. The Animal class should include a toString() method that returns a string in the format of 'name: {name}, age: {age}'. Then, create a Lion class that extends Animal and includes a roar() method returning 'roar!'. You also need to create a Fish class that extends Animal and includes a swim() method returning 'swim!'. For simulation, you will receive a string either 'Lion' or 'Fish' and you need to create an instance of the corresponding class, set its name to 'Simba' if 'Lion' and 'Nemo' if 'Fish', and its age to the integer value provided. Return the respective Animal's toString followed by respective method output. For example, if you received 'Lion' and the age 3, you should return 'name: Simba, age: 3, roar!'. Simulate the zoo using the provided animal type and age.
Sample Input:
Lion
3

Sample Output:
name: Simba, age: 3, roar!


Sample Input:
Fish
5

Sample Output:
name: Nemo, age: 5, swim!
