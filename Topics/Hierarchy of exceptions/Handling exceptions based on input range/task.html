Handling exceptions based on input range

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You are given a single integer input. You need to develop a simple application that treats inputs less than 0 as exceptions (throw new ArithmeticException()). If the value is equal or greater than 0 but less than or equal to 100, the program must throw an exception (throw new ArrayIndexOutOfBoundsException()). Finally, if the value is greater than 100, the application should print the input. If an exception occurs, print the exception's name.
Sample Input:
-15

Sample Output:
ArithmeticException


Sample Input:
50

Sample Output:
ArrayIndexOutOfBoundsException
